Half-Life: Resonance Cascade is a full conversion modification for SCP: Containment Breach
based on Half-Life game series. This mod serves as a depiction of the Black Mesa Incident
from a separate, farther off perspective.

You, Dr. Ben Walker, Clearance Level 4, have been assigned to test on BMR-500,
a robot that displays extreme hostility, but is fortunately only able to move while
outside direct observation. Midway into the testing session, you hear the alarm and the
scientists say they have it under control and it's not much of a problem. Turns out it is,
as the electricity shuts down and the robot kills your buddies and the guard. Sector C
Personnel have been ordered to evacuate the area until further notice. How far will you go
to get to the surface?

========================================= HOW TO RUN? =========================================

Run Half-Life Resonance Cascade.exe in the game folder. Select the resolution you want to
use and click the LAUNCH button.

======================================== BUGS & ERRORS ========================================

If you're getting an error such as "Memory Access Violation" or "Unable to set graphics mode"
when launching the game, try the following:

- Run the game as administrator
- Make sure your graphics card drivers are up-to-date
- Try different resolutions and running the game in full screen
- Try running the game in different compatibility modes

------------------------------------------ IMPORTANT! ------------------------------------------

Before playing, try to open 4GB Patch (4gb_patch.exe) in the "Source Code" folder and
choose "Half-Life Resonance Cascade.exe" in the pop-up screen - this should help you avoid
most of game crashes.

=========================================== CONTROLS ==========================================

	WASD				-	Move.
	Space				-	Blink.
	Left Shift			-	Sprint.
	Left Control			-	Crouch.
	Tab				-	Toggle inventory.
	F5				-	Save.
        Left mouse button               -       Interact (items, buttons, levers).
	F1				-	Take a screenshot.
	F3				-	Developer console (inactive by default).


Use items by double clicking them in the inventory.

Drop items by dragging them out of their inventory slot.

Combine certain items by dragging an item into the another inventory slot.
